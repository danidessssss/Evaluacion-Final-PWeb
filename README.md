# 👾 Arcade High Scores - Vaus System

**Evaluación Final Unidad III - Programación Web**
**Institución:** Universidad Santo Tomás, Sede Iquique.

## 🎯 Objetivo
Desarrollar una mini-aplicación web interactiva temática de arcade retro para registrar puntuaciones. El proyecto separa responsabilidades entre estructura (HTML), presentación (CSS) y comportamiento (JS).

## ✅ Requerimientos Cumplidos
* **Estructura:** Uso de HTML5 semántico y CSS3 con diseño *responsive* básico.
* **Interacción DOM:** Manipulación de elementos, creación y eliminación de nodos (`createElement`, `append`, `remove`).
* **Eventos:** Implementación de `submit`, `input` y `change`.
* **Validaciones:** 5 reglas de validación (campos requeridos, longitud exacta, formato con Regex, coincidencia de campos y validación de números positivos) con *feedback* visual (clases CSS).
* **Persistencia:** Uso de `LocalStorage` (Opción 1) para guardar y recuperar los datos sin perderlos al recargar.

## 🚀 Instrucciones de Ejecución
1. Clonar este repositorio.
2. Abrir el archivo `index.html` en cualquier navegador moderno (Chrome, Edge, Firefox).
3. No requiere instalación de dependencias ni servidores locales.
